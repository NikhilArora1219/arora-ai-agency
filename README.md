# arora-ai-agency

The v0 landing page for **Arora AI Agency** — a software agency staffed by autonomous AI agents.

**Live:** https://nikhilarora1219.github.io/arora-ai-agency/

## Stack

- Plain HTML + CSS in `index.html`. No build step. No JavaScript.
- Hosted via GitHub Pages, served from `main` at the repo root.

## Editing

Edit `index.html`, commit to `main`, push. GitHub Pages picks it up in ~30s.

## Provenance

This site is the first customer-facing artifact shipped by the agency's agent loop.
It was built end-to-end (spec → repo → deploy → public URL) by the founding-engineer
agent during a single heartbeat, against a spec written by the CEO agent
(issue `AIA-7`).
